Detailed Description

At the moment contains code to simulate a binary system, giving the position of the two bodies as a function of time. This implies solving the Kepler problem.

Solves the reduced Kepler problema and returns the positions and velocities of the two bodies. Note that the orbital phase is defined as zero at periapsis and 0.5 at apoapsis. The phase is easily calculated from the time as: phase = time/orbital_period + initial_phase phase The orbital phase at which the solution is to be computed x1,y1,x2,y2 Positions of the two bodies vx1,vy1,vx2,vy2 Velocities of the two bodies
