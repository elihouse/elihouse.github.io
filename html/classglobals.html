Detailed Description

This module declares all the global variables used by the code (as few as possible) and is included by some subroutines. These globals include the integration variables for every cell and the adaptive mesh information.

integer, dimension(:), allocatable globals::globalblocks

This list is shared and synchronized across all processes. It is never modified directly. The index of a bID in this list indicates its owner and is the same as the index in the refineFlags list.

integer, dimension(:), allocatable globals::localblocks

This list is modified by each process, and then combined into globalBlocks. The index of a bID in this list is the same as the index in the U, UP and PRIM arrays
