subroutine boundary ( integer, intent(in)  depth,
real, dimension(nbmaxproc, neqtot, nxmin:nxmax, nymin:nymax, nzmin:nzmax), intent(inout)  uvars 
)

This is a wrapper routine that calls the two boundary-passing routines: normalBoundary, which applies standard boundary conditions, and userBoundary, which applies user-defined custom conditions (this routine is provided by user.f90, a file which must be modified by the user on a per-problem basis).

Parameters
depthThe number of layers of ghost cells that are to be passed
uvarsVector of flow variables to be modified
