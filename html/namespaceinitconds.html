Detailed Description

This user-defined module establishes the parameters, variables and subroutines required to initialize the flow. How to use this module: 1) Specify all the parameters in the REQUIRED PARAMETERS section. These are all mandatory and are required for initflow to function. 2) Enter any aditional parameters or variables in the ADDITIONAL USER PARAMETERS section. 3) Provide a initflow subroutine, which will be called by initflow after refining the basegrid to the required resolution for the IC zone. The code expects your subroutine to modify the flow variables (U array) appropriately to impose the desired ICs.
