72  !! @n For the flow variables (U/UP), this is:
73  !! @li U(:,1,i,j,k) : @f$\rho@f$
74  !! @li U(:,2,i,j,k) : @f$\rho u@f$
75  !! @li U(:,3,i,j,k) : @f$\rho v@f$
76  !! @li U(:,4,i,j,k) : @f$\rho w@f$
77  !! @li U(:,5,i,j,k) : @f$E@f$ (kinetic+thermal)
78  !! @li U(:,6,i,j,k) : @f$B_x@f$
79  !! @li U(:,7,i,j,k) : @f$B_y@f$
80  !! @li U(:,8,i,j,k) : @f$B_z@f$
81  !! @li U(:,9+,i,j,k) : passive scalars
82  !! @li -OR-
83  !! @li U(:,6+,i,j,k) : passive scalars
